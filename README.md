# Kainos_LT

### Reikalavimai sistemai:
- Python 3.10 +
- Selenium

### Diegimas:
- pip install selenium
- pip install kainos_lt_scraper

### Naudojimas (command-line):
- kainos_lt_scraper.main
##### Argumentai:
- --time_limit (default: 120)
- --thread_count (default: 2)
- --max_thread_count (default: 5)
